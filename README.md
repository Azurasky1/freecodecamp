# javascript-snippets

## Installation

*   Install all the dependencies with `npm install`.

*   Run all the unit tests with `npm test`and generate an html report inside
    `mochawesome-reports`

**note**  All the unit tests are inside the `src/js` and marked with
the `*.spec.js` extension
